=======================
Portal Attendance Sync
=======================

Este módulo añade un botón de fichaje (entrada/salida) en la cabecera del portal de usuario para Odoo 18.

Configuración
=============

1. Asigne el grupo **“Permitir Fichaje en Portal”** a los usuarios.
2. Asegúrese de que el usuario tenga un registro vinculado en **hr.employee**.
